# Long Long Run

Long Long Run (LLR) is a plug-and-play Codex skill for serious long-running agent tasks. It provides an agent-first harness runtime with INC intent-noise cancellation, ACTIVE durable mainlines, side-thread tolerance, and Codex hook-enforced stop guards.

Use it when a Codex task needs more than a short edit-and-exit turn: project exploration, recovery, monitoring, long-running execution, or any objective that must stay coherent while side conversations continue.
